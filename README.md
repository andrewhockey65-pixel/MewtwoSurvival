# MewtwoSurvival

A p5.js survival game created for the AP CSP Create Project. Help Mewtwo survive endless attacks while collecting berries to maintain health!

## About the Game

In **MewtwoSurvival**, you control Mewtwo as it faces an onslaught of falling attacks from the sky. Your goal is to:
- **Avoid** enemy attacks falling from above
- **Collect** berries to restore and maintain Mewtwo's HP
- **Survive** as long as possible!

## How to Play

### Controls
- **Left Arrow Key** - Move Mewtwo left
- **Right Arrow Key** - Move Mewtwo right

### Gameplay Mechanics
- Dodge incoming attacks to avoid taking damage
- Collect falling berries to heal and increase your score
- Watch your HP bar - when it reaches zero, it's game over!
- The game gets progressively harder the longer you survive

2. Open `index.html` in your web browser

## How to Deploy Your Own Version

If you want to host your own version on GitHub Pages:

1. **Prepare Your Files**
   - Download your p5.js project files from the p5.js Web Editor as a .zip
   - Extract the files on your computer

2. **Create a GitHub Repository**
   - Log in to GitHub and click the + icon in the top right
   - Select "New repository"
   - Name your repository
   - Click "Create repository"

3. **Upload Your Files**
   - Click "Add file" → "Upload files"
   - Drag and drop your extracted files (including the libraries folder)
   - Scroll down and click "Commit changes"

4. **Enable GitHub Pages**
   - Go to the Settings tab
   - Click "Pages" in the left sidebar
   - Under "Build and deployment", select `main` (or `master`) as your branch
   - Leave the folder set to `/ (root)`
   - Click "Save"

Your game will be live at `https://yourusername.github.io/your-repo-name/`
